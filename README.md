# 🧶 Macramé Studio

Welcome to **Macramé Studio**, a handcrafted artisan project where every piece is designed and created by hand in Costa Rica.

This project combines traditional craftsmanship with modern web technologies, creating a digital storefront that showcases original handmade creations while experimenting with interactive web development.

---

# 🌿 About the Project

Macramé Studio is an independent artisan shop dedicated to creating handcrafted decorative pieces using traditional macramé techniques.

Every product is:

* 🧶 100% handmade
* 🇨🇷 Crafted in Costa Rica
* ✨ Designed and produced by the creator
* ♻️ Created with attention to quality and detail

Each piece is unique and reflects the creativity, patience, and craftsmanship behind handmade art.

---

# 💻 Technology

The website is also a web development project designed as part of the SkyCortex Creative Laboratory.

Current technologies include:

* HTML5
* CSS3
* JavaScript
* Responsive Design
* Google Sheets integration
* Lightweight CRM system

---

# 📋 CRM System

The current version includes a lightweight customer management system connected to Google Sheets.

The CRM is designed to:

* Store customer inquiries
* Collect contact requests
* Organize product information
* Simplify order management
* Support future business automation

This system is currently under active development and will continue expanding with additional features.
